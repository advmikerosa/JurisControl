# Configuração do Banco de Dados (Supabase)

O setup do banco de dados agora está consolidado no arquivo `schema.sql` para evitar erros de sintaxe ao copiar e colar.

## Instruções

1. Abra o arquivo `schema.sql` na raiz do projeto.
2. Copie todo o conteúdo.
3. Vá para o Painel do Supabase > **SQL Editor**.
4. Cole o código e clique em **Run**.

Isso criará todas as tabelas, funções e políticas de segurança (RLS) necessárias para o funcionamento do **JurisControl**.
